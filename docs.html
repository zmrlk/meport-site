What is Meport?

Meport is a CLI tool that creates a comprehensive AI personality profile through a single conversation. Your AI doesn't know you — Meport fixes that in 30 minutes.

It generates personalized instructions for ChatGPT, Claude, Cursor, Copilot, Windsurf, Ollama, Gemini, Grok, Perplexity and more — all from one profile.

How it works 1. Answer questions (or let AI interview you) → 2. Meport builds a profile with 30+ dimensions → 3. Export to any AI platform → 4. Your AI finally knows how to talk to you.

Installation

Meport requires Node.js 18+. No installation needed — run directly with npx:

Terminal
npx meport

Or install globally:

Terminal
npm install -g meport
meport
Tip Running meport with no arguments opens the interactive shell — a menu-driven interface for all features.

Your First Profile

The fastest way to get started:

1

Run meport profile. If AI is configured, it starts an AI-driven interview. Otherwise, it uses the quiz mode.

2

Answer the questions. Meport auto-detects your system (OS, timezone, tools, languages) and skips what it already knows.

3

Review your profile and confirm. Meport auto-exports to all platforms and shows you how to apply each one.

Terminal
# AI-driven (best quality, needs API key)
meport profile --ai

# Quick quiz mode (no API key needed)
meport profile --quick

# With file scanning
meport profile --scan ~/Projects ~/Documents/resume.pdf

# Polish language
meport profile --lang pl

meport profile

Start profiling — one conversation, full AI profile. This is the main command. It runs system scan, asks questions via packs, shows a summary, and auto-exports to all platforms.

Syntax
meport profile [options]

Options

Flags
-o, --output <path>       Output file path (default: ./meport-profile.json)
-e, --export-dir <path>  Auto-export directory
-l, --lang <locale>      Language for questions (en, pl)
-s, --scan <paths...>    Scan files or folders for profile data
-q, --quick              Quick mode — key questions only, instant profile
    --ai                 Force AI-driven conversational profiling

Examples

Terminal
# Standard profiling (auto-detects AI config)
meport profile

# AI-driven interview
meport profile --ai

# Quick mode — 5 minutes, core questions only
meport profile --quick

# Scan local files for extra context
meport profile --scan ./resume.pdf ./portfolio/

# Polish language, custom output
meport profile --lang pl -o ~/my-profile.json

# Full pipeline: scan + export to specific directory
meport profile --scan ~/Projects --export-dir ~/ai-configs/
Profiling flow Phase 0: System scan (silent) → Phase 1: Intro → Phase 2: File scan (if --scan) → Phase 3: Pack questions → Phase 4: Final summary + AI rules preview → Phase 5: Auto-export to all platforms. Press Ctrl+C at any time — session is auto-saved. Resume with meport continue.

meport export

Export your profile to a specific platform format. Uses rule-based compilers by default (14 platforms). Falls back to legacy compilers with --legacy.

-c, --copy              Copy export to clipboard (single platform)

Terminal

meport export --all -o ./my-exports/

meport export chatgpt --legacy

Question Packs

Packs are modular question sets that target specific areas of your personality and preferences. During profiling, you choose which packs to answer. You can add or remove packs at any time.

Each pack contains 5-10 scenario-based questions with multiple choice answers. Questions are designed to reveal behavioral patterns, not just preferences.

All Packs

micro-setupAuto-loaded. Name, intro, pack selection. Always runs first.
coreCommunication DNA — how you like AI to talk to you. Verbosity, directness, anti-patterns, feedback style.
contextLife Context — job, location, current focus, tools, industry.
workWork Style — energy patterns, deadline behavior, perfectionism, task preferences.
lifestyleLifestyle — daily rhythm, social energy, routine preference, travel, food.
healthHealth (sensitive) — fitness level, sleep patterns, conditions. Data is privacy-filtered.
financeFinance (sensitive) — money style, budgeting, financial goals. Data is privacy-filtered.
learningLearning — how you learn, study preferences, format preference, time commitment.
Sensitive packs Health and Finance packs are marked as sensitive. They show privacy notes before questions and their data goes through privacy filtering in exports. You can skip individual questions or entire packs.

Scanning

Meport can auto-detect information about you from your system and files. This happens automatically during profiling (Phase 0) and can also be triggered manually.

System scan (automatic)

Runs silently at the start of every profiling session. Detects:

  • Operating system, timezone, locale
  • Username / preferred name (from git config or OS)
  • Installed tools (editors, languages, frameworks)
  • Tech stack (from package.json, Cargo.toml, etc.)
  • AI tools in use (Cursor, Claude Code, Copilot, etc.)

File scan (opt-in via --scan)

Scan specific files or directories for additional profile data. Useful for resumes, portfolios, project files.

Terminal
# Preview what meport can detect
meport scan ~/Projects ~/resume.pdf

# Use detected data during profiling
meport profile --scan ~/Projects ~/resume.pdf

Privacy filter

Sensitive data (health conditions, financial details) is marked as sensitive and goes through privacy filtering before export. Dimensions starting with _ are internal and never exported. You can review all detected dimensions with meport view.

AI Providers

Meport uses AI for three features: conversational profiling (--ai), deepening sessions, and report generation. Configure via meport config.

Claude (Anthropic)

Best quality. Requires an Anthropic API key. Default model: claude-sonnet-4-20250514.

# Enter your API key
# Model: claude-sonnet-4-20250514 (default)

Good quality. Requires an OpenAI API key. Default model: gpt-4o-mini.

Setup
meport config
# Select: OpenAI (GPT)
# Enter your API key
# Model: gpt-4o-mini (default)

Ollama (local)

Free, runs on your machine. No API key needed. Default model: llama3.1. Default URL: http://localhost:11434.

Setup

ollama pull llama3.1

meport config

# URL: http://localhost:11434
# Model: llama3.1
Without AI Meport works fine without any AI provider. Quiz mode (meport profile) uses pre-built question packs with multiple choice answers. AI just unlocks conversational interviews, deeper reports, and smarter gap-filling.

FAQ

Do I need an API key to use meport?

No. Quiz mode works without any API key. AI features (conversational interview, deepening, reports, demo) require an API key from Claude, OpenAI, or a local Ollama installation.

Is my data sent anywhere?

Your profile stays on your machine in meport-profile.json. If you use AI features, your answers are sent to the configured AI provider (Claude/OpenAI/Ollama). With Ollama, everything stays local.

How long does profiling take?

Quick mode: 5 minutes. Full profiling: 15-30 minutes. AI-driven interview: 10-20 minutes. You can interrupt at any time (Ctrl+C) and resume later with meport continue.

Can I use meport for work and personal profiles?

Yes. Use -o to save to different files: meport profile -o work-profile.json and meport profile -o personal.json. Then export each separately.

How often should I refresh my profile?

Run meport refresh every 2-4 weeks, or whenever something significant changes (new job, new tools, new goals). Meport will detect changes and ask for confirmation.

What happens if I skip questions?

Skipped questions reduce profile completeness but don't break anything. You can always come back later with meport deepen or meport packs add <pack> to fill gaps.

Can I manually edit the profile JSON?

Yes, but use meport edit instead — it validates changes and auto-saves snapshots. If you edit the JSON directly, run meport refresh afterward to recompute derived signals.

What is the difference between deploy and export?

meport export generates files. meport deploy generates files AND writes them to the correct locations in your project (Cursor rules, CLAUDE.md, etc.). Deploy is the recommended workflow.

How does the smart merge work for CLAUDE.md?

When deploying to a file that already has content, meport appends a # --- meport profile --- section. On subsequent deploys, only that section is replaced. Your custom content above it is preserved.

Which AI provider is best?

Claude (Anthropic) gives the best quality for interviews and reports. GPT-4o-mini is a good budget option. Ollama is free but quality depends on the model. For profiling without AI, quiz mode is excellent.
