How it works

Privacy

Your data stays yours.

0

Servers

We don't have servers. Your profile lives in a local JSON file on your machine. There's nowhere to send it.

3K

Chars to AI

When AI helps, each file sends at most 3,000 characters after privacy filtering. Files are read locally up to 5K — but AI never sees more than 3K.

11

Patterns blocked

Passwords, API keys, PESEL, NIP, IBAN, credit cards, private keys, tokens — auto-redacted. Can't be disabled.

0

Telemetry

No analytics. No session IDs. No tracking pixels. No DNS pings. Ollama mode — zero network requests, period.

CV & ResumeBookmarksFoldersNotesCalendarLinkedIn

We read traits, not content.

Your files are scanned locally for structured signals only. A resume gives us "React, 5 years" — not your address. Bookmarks reveal tools you use — not browsing history. Folders show your tech stack — not file contents. Nothing is indexed or stored as raw text.

Sent to your AINever sent

AI is optional. Here's exactly what it sees.

Sent: your interview answers ("I like direct feedback"), extracted dimensions ("prefers bullet points"), and up to 3K chars per file — after the privacy filter. Never sent: full file contents, passwords, financial data, identity numbers, file paths. Data goes to YOUR provider (Claude, OpenAI, Gemini, Grok, OpenRouter). Meport is never a middleman. Choose Ollama — nothing leaves localhost.

Open source. Every line that touches your data is on GitHub. Read the source — we built it this way because we use it ourselves.
